### 概述
`jsmod` 目标提供高自由度（高扩展性）的组件集合，并统一桌面端、移动端的使用方式，是一个面向**用户端**的组件库


### 文档
[点击查看](http://jsmod-vue.tedfe.com)

### 安装和使用
```
npm install jsmod-pc-vue --save

```


```javascript
import Vue from 'vue'
import JSMOD from 'jsmod-pc-vue'

Vue.use(JSMOD)
```
